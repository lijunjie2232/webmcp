# WebMCP Documentation

Welcome to WebMCP documentation. WebMCP is a powerful MCP (Model Context Protocol) server that provides advanced search capabilities using multiple search engines with deep content extraction.

## Quick Links

- [Installation Guide](installation.md) - Get started with WebMCP
- [Usage Guide](usage.md) - Learn how to use WebMCP
- [Development Guide](development.md) - Contribute and extend WebMCP

## Features

- **Multi-Engine Support**: Search across Google, Bing, DuckDuckGo, and Yahoo Japan
- **Deep Search**: Automatically parse search result pages to extract complete content
- **Language Localization**: Set any language/locale for region-specific search results
- **Parallel Processing**: Efficiently parse multiple pages simultaneously
- **Resource Optimization**: Smart resource blocking for faster page loads
- **MCP Compatible**: Easy integration with LangChain and other MCP clients

## Supported Search Engines

- ✅ Google
- ✅ Bing
- ✅ DuckDuckGo
- ✅ Yahoo Japan

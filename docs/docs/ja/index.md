# WebMCP ドキュメント

WebMCPドキュメントへようこそ。WebMCPは、複数の検索エンジンを使用して高度な検索機能と深いコンテンツ抽出を提供する強力なMCP（Model Context Protocol）サーバーです。

## クイックリンク

- [インストールガイド](installation.md) - WebMCPを使い始める
- [使用方法](usage.md) - WebMCPの使用方法を学ぶ
- [開発ガイド](development.md) - WebMCPに貢献・拡張する

## 主な機能

- **複数エンジン対応**: Google、Bing、DuckDuckGo、Yahoo Japanで検索
- **ディープサーチ**: 検索結果ページを自動的に解析して完全なコンテンツを抽出
- **言語ローカライゼーション**: 地域固有の検索結果のために任意の言語/ロケールを設定
- **並列処理**: 複数のページを効率的に同時に解析
- **リソース最適化**: 高速なページ読み込みのためのスマートなリソースブロック
- **MCP互換**: LangChainおよび他のMCPクライアントとの簡単な統合

## サポートされている検索エンジン

- ✅ Google
- ✅ Bing
- ✅ DuckDuckGo
- ✅ Yahoo Japan
